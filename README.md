# TP3-HTML

Exercice de mise en page réalisé dans le cadre de ma formation Développeur Web.

## Objectif

Reproduire fidèlement une maquette fournie en HTML et CSS pur, puis rendre la page entièrement responsive sur mobile, tablette et desktop.

## Compétences travaillées

- Intégration pixel-perfect d'une maquette
- Structuration sémantique du HTML
- Mise en page avec Flexbox et/ou CSS Grid
- Responsive design avec les media queries
- Organisation et lisibilité du code CSS

## Stack technique

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

Aucune librairie externe, aucun framework : HTML et CSS natifs uniquement.
